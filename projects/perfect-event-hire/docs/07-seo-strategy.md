# 07 – SEO Strategy

## Purpose

This document outlines the search engine optimisation strategy implemented to improve the visibility of Perfect Event Hire within Google Search.

---

# SEO Objectives

The strategy aimed to:

- Improve organic search rankings.
- Increase website traffic.
- Improve local search visibility.
- Generate more enquiries.

---

# Keyword Strategy

Primary keywords included:

- Event Hire Sydney
- Wedding Hire Sydney
- Corporate Event Hire
- Party Hire Sydney
- Event Styling Sydney

Secondary keywords included:

- Wedding Decorations
- Balloon Styling
- Event Furniture Hire
- Backdrop Hire
- Party Packages

---

# On-Page SEO

Optimisations included:

- Meta titles
- Meta descriptions
- Heading structure (H1–H3)
- Image ALT text
- Internal linking
- Content optimisation

---

# Local SEO

Focus areas:

- Google Business Profile
- Sydney service areas
- Location-based keywords
- Consistent business information

---

# Technical SEO

Implemented improvements:

- Mobile responsiveness
- Optimised images
- Faster page loading
- Clean navigation
- Crawl-friendly structure

---

# Content Strategy

Content was written to:

- Answer customer questions.
- Improve keyword relevance.
- Increase engagement.
- Support conversions.

---

# Expected Benefits

- Higher rankings
- More website traffic
- Better customer engagement
- Increased enquiries
