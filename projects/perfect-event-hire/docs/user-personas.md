# 05 – User Personas

## Purpose

Understanding different customer types helps ensure the website is designed around real user needs rather than assumptions.

---

# Persona 1 – Bride Planning a Wedding

## Goals

- Find wedding styling inspiration.
- View available products.
- Request a quotation.

## Pain Points

- Too many clicks.
- Difficult navigation.
- Limited gallery images.

## Needs

- Beautiful imagery.
- Clear pricing information.
- Simple enquiry process.

---

# Persona 2 – Corporate Event Organiser

## Goals

- Find reliable suppliers.
- View professional services.
- Receive a fast response.

## Pain Points

- Poor website credibility.
- Limited business information.

## Needs

- Professional branding.
- Easy contact.
- Service information.

---

# Persona 3 – Birthday Event Customer

## Goals

- Browse decorations.
- Compare packages.
- Book quickly.

## Pain Points

- Hard-to-find products.
- Confusing navigation.

## Needs

- Easy browsing.
- Mobile-friendly website.
- Clear gallery.

---

# Common User Needs

Across all personas, users expect:

- Fast loading pages.
- Mobile responsiveness.
- Professional appearance.
- Clear navigation.
- Simple enquiry forms.

---

# Design Considerations

The website should:

- Reduce unnecessary clicks.
- Use clear headings.
- Present services visually.
- Build trust immediately.
