# 06 – Site Architecture

## Purpose

This document defines the logical structure of the Perfect Event Hire website. A well-organised site architecture improves navigation, user experience and search engine optimisation.

---

# Objectives

The website architecture was designed to:

- Help visitors locate information quickly.
- Improve website usability.
- Support search engine indexing.
- Create a scalable structure for future growth.
- Reduce unnecessary navigation.

---

---

# Navigation Principles

The navigation should:

- Be simple.
- Be consistent.
- Require minimal clicks.
- Highlight important services.
- Encourage enquiries.

---

# Internal Linking

Each service page should connect to:

- Gallery
- Related services
- Contact page
- Quote request

This strengthens both SEO and user navigation.

---

# SEO Benefits

A clear site architecture helps search engines:

- Crawl pages efficiently.
- Understand relationships between pages.
- Improve indexing.
- Strengthen internal authority.

---

# Expected Outcomes

- Easier navigation
- Better customer experience
- Improved search rankings
- Higher enquiry conversion
