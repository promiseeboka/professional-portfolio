# 07 – UX Improvements

## Purpose

This document explains the user experience improvements made throughout the Perfect Event Hire website.

The objective was to create a website that is intuitive, visually engaging and easy to navigate.

---

# UX Objectives

- Reduce customer frustration.
- Improve navigation.
- Increase enquiries.
- Improve mobile usability.
- Create a professional experience.

---

# Improvements

## Navigation

- Simplified menu
- Clear page hierarchy
- Logical grouping of services

---

## Homepage

Improvements included:

- Cleaner layout
- Stronger hero section
- Improved call-to-action
- Better service overview

---

## Service Pages

Updated with:

- Better descriptions
- High-quality images
- Clear enquiry buttons
- Improved readability

---

## Gallery

Enhanced by:

- Better organisation
- Larger images
- Improved mobile viewing

---

## Contact Page

Optimised to:

- Reduce form complexity
- Improve accessibility
- Encourage customer enquiries

---

# Mobile Experience

Improvements included:

- Responsive layouts
- Larger buttons
- Easier scrolling
- Faster loading

---

# UX Principles Applied

- Simplicity
- Consistency
- Visibility
- Accessibility
- User-centred design

---

# Expected Outcomes

- Better customer satisfaction

- More enquiries
- Improved usability
