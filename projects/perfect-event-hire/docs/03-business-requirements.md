# 03 – Business Requirements

## Purpose

This document defines the business requirements for the redesign and optimisation of the Perfect Event Hire website. The requirements were identified to ensure the website supports business objectives, improves the customer experience and provides a strong foundation for future digital growth.

---

# Business Goals

The website should:

- Increase customer enquiries.
- Improve the online visibility of the business.
- Present services professionally.
- Improve customer trust and credibility.
- Support long-term business growth.
- Strengthen the company's digital marketing efforts.

---

# Functional Requirements

## Website Navigation

The website must:

- Provide a clear navigation menu.
- Allow visitors to locate services quickly.
- Display contact information consistently.
- Maintain consistent navigation across all pages.

---

## Service Information

The website must:

- Display all event hire services.
- Include detailed service descriptions.
- Showcase available products.
- Present high-quality project images.

---

## Contact & Enquiries

Visitors must be able to:

- Submit enquiries through an online form.
- Request quotations.
- Contact the business using multiple communication methods.

---

## Gallery

The website should:

- Display professional event photography.
- Organise images into logical categories.
- Support mobile-friendly image viewing.

---

## Mobile Experience

The website must:

- Display correctly on mobile devices.
- Adapt to different screen sizes.
- Maintain usability across all devices.

---

# SEO Requirements

The website should support:

- Optimised page titles.
- Meta descriptions.
- Image ALT text.
- Internal linking.
- Structured page headings.
- Local SEO.
- Fast page loading.

---

# Performance Requirements

The website should:

- Load quickly.
- Optimise image sizes.
- Minimise unnecessary page elements.
- Provide a smooth browsing experience.

---

# Branding Requirements

The website should:

- Present a modern appearance.
- Maintain consistent colours.
- Use professional typography.
- Reflect the Perfect Event Hire brand.
- Build customer confidence.

---

# User Requirements

Customers should be able to:

- Understand available services quickly.
- Browse products easily.
- Find contact information without difficulty.
- Request quotations with minimal effort.

---

# Success Criteria

The project will be considered successful if it delivers:

- Improved user experience.
- Better website organisation.
- Stronger SEO foundations.
- Increased enquiry opportunities.
- Improved mobile usability.
- Professional digital presence.
